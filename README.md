**Student Information System**
**Project Overview**

A Java console-based application to manage student records using a menu-driven system.<br />

**Objectives**

Store and manage student details <br />
Perform add, view, update, delete, and search operations <br />
Validate user input and display formatted output <br />

**Tech Stack** <br />
Java <br />
ArrayList<br />
Scanner<br />

**Project Structure**

model – Student class<br />
service – Business logic<br />
controller – Menu and user actions<br />
main – Program entry point<br />

**Features**

Add, view, update, delete students<br />
Search by student ID<br />
Input validation<br />
Clean console output<br />

**How to Run**

Open project in IntelliJ/Eclipse<br />
Run the Main class<br />
Use menu options<br />
