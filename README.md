Student Information System
Project Overview

A Java console-based application to manage student records using a menu-driven system.

Objectives

Store and manage student details

Perform add, view, update, delete, and search operations

Validate user input and display formatted output

Tech Stack

Java

ArrayList

Scanner

Project Structure

model – Student class

service – Business logic

controller – Menu and user actions

main – Program entry point

Features

Add, view, update, delete students

Search by student ID

Input validation

Clean console output

How to Run

Open project in IntelliJ/Eclipse

Run the Main class

Use menu options
